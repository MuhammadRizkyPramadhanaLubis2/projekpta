# Evaluasi AKIP Binjai - Page Baru

Prototype page PHP untuk menampilkan isi PDF "Lembar Kerja Evaluasi Akuntabilitas Kinerja Pengadilan Agama Binjai Tahun 2023" dalam satu halaman web.

## Menjalankan

```bash
php -S 127.0.0.1:8080 -t .
```

Buka:

```text
http://127.0.0.1:8080/index.php?page=evaluasi-akip-binjai
```

## Catatan

- Bahasa yang dipakai tetap PHP agar konsisten dengan repo `projekpta`.
- Desain dibuat bergaya emerald-gold, glassmorphism, dan kartu informasi seperti nuansa portal IKPA.
- Struktur data mengikuti isi file PDF yang diberikan user.
